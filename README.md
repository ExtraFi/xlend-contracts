# Extra X Contracts

This repository contains the smart contracts for ExtraFi XLend, which integrates a SmartAccount with a lending protocol.

### Forked Repositories

- **contracts/core-v3**: This directory is forked from the [Aave V3 Core](https://github.com/aave/aave-v3-core.git) repository. The forked commit is [b74526a7bc67a3a117a1963fc871b3eb8cea8435](https://github.com/aave/aave-v3-core/commit/b74526a7bc67a3a117a1963fc871b3eb8cea8435).

### ExtraX Smart Account

- **contracts/extra-x-account/ExtraXAccountFactory.sol**: This contract implements the factory for creating Smart Accounts.
